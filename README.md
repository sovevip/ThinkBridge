# ThinkBridge — AI 思维框架教练

> 不是替你思考，而是训练你学会思考。

手机浏览器打开就能用的 PWA。输入一个问题，AI 自动判断问题类型、选择合适的思维框架，一步步教你分析，最后给你可复用的自我提问模板。

## 快速开始

1. 打开 `index.html`（或部署到 GitHub Pages）
2. 在设置中填入你的 DeepSeek API Key
3. 输入一个困扰你的问题，开始分析

## 功能

- 📖 学习模式：完整教学型分析报告，多框架交叉验证
- ⚡ 快速分析模式：精简判断 + 行动建议
- 📚 框架库：浏览 32 个思维框架，搜索、分类、对话学习
- 📋 历史记录：保存、回看、复制过往报告
- 📱 PWA：可添加到手机桌面，离线可浏览框架库和历史

## 技术

单文件 HTML + Tailwind + Alpine.js。浏览器直连 DeepSeek API。API Key 和历史记录仅存本地。

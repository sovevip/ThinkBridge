# ThinkBridge — 产品需求文档（PRD）

## 1. 产品概述

ThinkBridge 是一个本地自用的移动端 PWA，也是一名 AI 思维框架教练。用户输入真实问题后，AI 自动判断问题类型、选择合适的思维框架，一边示范如何分析，一边解释为什么这样思考，最后给出可复用的自我提问模板。

- 核心目标：不是替用户思考，而是训练用户学会思考
- 目标用户：希望提升结构化思维能力的个人用户
- 产品形态：单文件 HTML PWA，手机浏览器打开即用

## 2. 核心功能

### 2.1 功能模块

1. **分析页**：输入问题 → AI 两步 Pipeline → 教学型分析报告
2. **框架库页**：浏览 32 个思维框架，分类筛选、搜索、查看详情
3. **历史页**：查看历史分析报告，支持回看、删除、清空

| 页面名称 | 模块名称 | 功能描述 |

## 3. 核心流程

用户打开页面 → 输入 API Key → 选择模式 → 输入问题 → 点击分析 → Step1 Router 诊断问题类型并选择框架 → 如需补充信息则展示追问 → Step2 Analyzer 流式输出教学型报告 → 保存历史 → 用户复制或回看。

- 主色调：深墨蓝 (#1a1a2e) + 暖金 (#e8b931) 点缀，营造沉稳思考氛围

## 5. 两步 Pipeline AI 链路

### Step 1：Router（路由与诊断）

- 输入：用户问题 + 框架简表 + 现实约束触发规则
- 输出：JSON（status / problem_type / selected_frameworks / validation_frameworks / reality_filters / reason）
- 信息不足时返回 status=need_more_info + 追问

### Step 2：Analyzer（分析与教学）

- 输入：用户问题 + Router JSON + 选中框架详细定义 + 现实约束规则 + 教学模板
- 输出：流式 Markdown 教学型报告

### JSON 解析兜底

5 级策略：直接解析 → 提取代码块 → 正则提取对象 → 修复重试 → 降级默认框架

## 6. 隐私说明

- API Key 和历史记录保存在本地浏览器
- 问题会发送给 DeepSeek API
- 不主动要求敏感信息
- 提醒用户避免输入身份证、银行卡等

## 7. 第一版成功标准

手机打开 → 输入真实问题 → AI 教我为什么用这个框架 → 看懂每一步 → 得到可复用的提问模板。
